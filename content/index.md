# The GET

</div>

**The GET (The Game Engine Tutor)** helps you take an idea for a playable story or world and shape it into a build order for a game engine. It's two things at once: a **tutor you talk to**, and a **library of course material** it teaches from.

## ▸ The GET — Web Version

The GET's corpus — what the tutor teaches from, browsable online:

- **Design** — [Storytelling: The Situated Player Roles](https://vlabusc.github.io/The-GET/Design/Storytelling/) · [Worldbuilding: Bounded Worlds](https://vlabusc.github.io/The-GET/Design/Worldbuilding/) · [References](https://vlabusc.github.io/The-GET/Design/_References/)
- **Development** — [Unreal tutorials](https://vlabusc.github.io/The-GET/Development/Tutorials---Unreal/) · [Unreal wiki](https://vlabusc.github.io/The-GET/Development/Wiki---Unreal/) · [AI-tool setup](https://vlabusc.github.io/The-GET/Development/Tutorials---LLM/)
